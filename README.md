# EletroBásica

Projeto para iniciantes que buscam adquirir conhecimento em eletrônica e elétrica usando HTML, CSS e JavaScript.

## Estrutura

- `index.html`: índice principal, fundamentos, artigos de apoio e indicações de faculdades/cursos próximos a Manoel Ribas-PR.
- `multimetro.html`: guia aprofundado de multímetro, escalas, bornes, procedimentos e erros críticos.
- `componentes.html`: guia de componentes eletrônicos, falhas comuns e testes iniciais.
- `testes.html`: roteiros práticos de bancada e modelo de registro de diagnóstico.
- `style.css`: estilos compartilhados entre todas as páginas.
- `script.js`: navegação ativa e botão de voltar ao topo.
